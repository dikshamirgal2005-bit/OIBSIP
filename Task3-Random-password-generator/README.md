Advanced Password Generator (Python + Tkinter)

This project is an advanced password generator built using Python and Tkinter.  
It allows users to generate secure passwords with multiple customization options and a simple graphical interface.

---

Features

- Choose password length  
- Select character types:
  - Uppercase Letters
  - Lowercase Letters
  - Numbers
  - Symbols
- Strong password generation rules  
- Clipboard copying  
- User-friendly Tkinter GUI  
- Input validation (prevents invalid length or empty input)
