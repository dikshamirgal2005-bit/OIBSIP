Advanced Weather App (Python + Tkinter)

This is an advanced Weather Application built using **Python**, **Tkinter**, and the **OpenWeatherMap API**.  
It allows users to:

✔ Search weather by city name  
✔ View temperature, humidity, wind speed  
✔ View weather condition + icon  
✔ See detailed description  
✔ Auto unit conversion (°C / °F)  
✔ Works on Windows, Linux, Mac  

---

Features

- Beautiful Tkinter GUI  
- Real-time weather using OpenWeather API  
- Displays:
  - Temperature
  - Feels-like temperature
  - Humidity
  - Wind Speed
  - Weather Description
  - Weather Icon  
- Error handling for:
  - Invalid city name
  - No internet
  - Incorrect API Key
